# Python Static Site Generator

![Python](https://img.shields.io/badge/python-3.9-blue.svg)
![Status](https://img.shields.io/badge/status-complete-brightgreen)

A simple, command-line static site generator built with Python. This script reads Markdown files, converts them to HTML, and injects them into a base template to create a complete static website.

## Features

-   Converts Markdown files to HTML pages.
-   Uses the powerful Jinja2 templating engine.
-   Simple directory structure for content and templates.
-   Customizable HTML base template.
-   Command-line interface for easy use.

## How to Use

### 1. Prerequisites

-   Python 3.6+
-   `pip` for installing packages

### 2. Installation

Clone the repository and install the required dependencies:

```bash
git clone https://github.com/krittaphato3/static-site-generator.git
cd static-site-generator
pip install -r requirements.txt
